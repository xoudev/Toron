Toron — le système de marque

Deux concepts, deux rôles — sans fusion. B « les strates » est la marque : elle vit dans le produit. C « le poinçon » est le sceau : il est frappé sur ce que le produit génère, avec l'empreinte du document. Le sceau devient une fonctionnalité d'intégrité, pas un ornement.
